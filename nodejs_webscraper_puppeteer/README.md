# Node webscraper with puppeteer

## Install dependencies
```bash
 "dependencies": {
    "puppeteer": "^15.3.0"
  }
```

1. Run this command
```bash
npm i
```

## Run the server

2. Run this command

```bash
npm start
```
or

```bash
node index.js
```

## Check the data.seed.js created

3. After the webpage is scraped, it creates a file ./data/data.seed.json with the data scraped
